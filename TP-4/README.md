# notes_app-CI_CD

## Installation

### Prérequis

- Docker et Docker Compose
- Node.js 18
- npm

### Configuration

Depuis le dossier `TP-4`, créer le fichier d'environnement à partir de l'exemple :

```bash
cp .env.example .env
```

Puis ajuster les variables si nécessaire.

### Lancement

Depuis le dossier `TP-4` :

```bash
docker compose up --build
```
